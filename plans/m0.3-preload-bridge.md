# M0.3 — Generic typed preload bridge (invoke + subscribe)

## Goal
Replace the hardcoded `send` channel in preload with a generic `ipc` object exposing:
- `invoke(channel, payload)` — request/response via `ipcRenderer.invoke`
- `subscribe(topic, cb)` — event subscription with wildcard support, returns unsubscribe fn

After this change, preload never needs editing again for new IPC channels.

## Changes
- **packages/preload/src/index.ts**: Replace `send` export with generic `ipc` object (`invoke` + `subscribe`)
  - Import `EVENT_CHANNEL` from `@app/core`
  - Keep `sha256sum` and `versions` exports for backwards compatibility
- No other files modified

## Validation
- `npm run build` — all workspaces compile cleanly
- `npm run screenshot` — app boots without errors
